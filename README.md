## Github flow
1.Clone the repo.  
2.Run npm install (or just npm i)  
3.Run npm start  
## Technologies
Project is created with:
* HTML
* CSS
* JavaScript
* TypeScript
* React
* React Router
* Hooks

## Features
* Search Cities, Hotels
* Choose Date Range
* Choose People and Kids Size
